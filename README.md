# Regexy

Regexy is the ruby gem that contains a lot of common-use regular expressions and provides a friendly syntax to combine them.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'regexy'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install regexy
